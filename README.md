# Netflix_Task1
# 🎬 Netflix Movies Data Analysis (Python | Pandas)

## 📌 Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on a dataset of ~10,000 Netflix movies. The goal is to clean, transform, and analyze the dataset to extract meaningful insights about movie trends, genres, and popularity.

---

## 🚀 Key Features

✔️ Data Cleaning & Preprocessing  
✔️ Handling Missing Values  
✔️ Data Type Conversion (Datetime, Numeric)  
✔️ Feature Engineering (Extracting Release Year)  
✔️ Genre Transformation (Splitting & Exploding)  
✔️ Categorization of Movies based on Ratings  
✔️ Insightful Analysis & Summary  

---

## 🛠️ Technologies Used

- Python 🐍  
- Pandas  
- NumPy  
- Jupyter Notebook  

---

## 📊 Data Preprocessing Steps

- Converted `Release_Date` into datetime format  
- Fixed incorrect data types for `Vote_Count` and `Vote_Average`  
- Removed rows with missing values  
- Extracted `Release_Year` from date  
- Dropped unnecessary columns like `Overview`, `Poster_URL`, `Original_Language`  
- Split multiple genres into individual rows using `explode()`  

---

## 📈 Key Insights

- 🎭 **Most Frequent Genre:** Drama (appears ~14% of total dataset)  
- ⭐ **Highest Popular Movie:** Spider-Man: No Way Home  
- 📉 **Lowest Popular Movies:** Threads & The United States vs Billie Holiday  
- 📅 **Year with Most Movies Released:** 2020  
- 🔥 Popularity categories created:  
  - Not Popular  
  - Below Average  
  - Average  
  - Popular
